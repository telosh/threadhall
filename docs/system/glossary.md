# 用語集

| 用語 | 定義 |
|------|------|
| **プライマリ DB** | 組織・identity・長命スレッド・イベントメタを載せる Turso |
| **サテライト DB** | 特定イベント専用の Turso。ログや当日データの隔離に使う |
| **アーカイブ** | イベントフェーズ `archived`。書込不可・閲覧中心 |
| **参加トークン** | 書込権限の主要根拠。QR や短命 JWT で表現 |
| **コントロールプレーン** | 「どのイベントがどのサテライト URL／認証を持つか」のメタを保持する層（通常はプライマリ内テーブルと運用 API） |
